
## 技术栈

vue2 + vuex + vue-router + webpack + ES6/7 + less + element-ui


## 项目运行


```
npm install 

npm install --save axios

npm install vue-quill-editor

npm run dev (访问线上后台系统)

npm run local (访问本地后台系统，需运行node-elm后台系统)


访问: http://localhost:8002

```
